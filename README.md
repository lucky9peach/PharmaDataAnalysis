# 数据分析市场研究工具

这是一个用于医药数据下载、清洗、标准化以及可视化的综合性工具系统。

## 准备工作 (环境安装与使用配置)

### 1. 下载代码库
使用 Git 克隆本仓库到本地环境：
```bash
git clone <这里替换为您在 GitHub 上创建的仓库地址>
cd 数据分析市场研究
```

### 2. 创建并激活虚拟环境 (强烈推荐)
为了隔离项目依赖，避免与系统中其他的 Python 库产生冲突，建议使用 Python 的 `venv`。

**Mac/Linux 环境:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows 环境:**
```cmd
python -m venv venv
venv\Scripts\activate
```

### 3. 安装依赖包
在激活的虚拟环境中，使用 pip 安装本项目所需要的所有第三方库：
```bash
pip install -r requirements.txt
```

### 4. 账号配置 (首次必做)
为了保护数据安全，账号密码不会直接写在代码中。请复制模板文件配置您的账号：
1. 在项目根目录下，找到 `config.example.ini`。
2. 将其复制或重命名为 `config.ini`。
3. 打开 `config.ini`，在其中填入您的 TSM 数据库下载账号和密码。
> 注意：`config.ini` 已经被 `.gitignore` 忽略，绝对不会被上传到共有服务器。

## 使用说明

运行项目的主入口是 `main.py`。该文件将引导启动整个图形用户界面（GUI），串联所有的分析步骤（自动下载 -> 数据清洗 -> 标准化 -> 可视化看板查看）。

在终端（或命令行）中，确保虚拟环境已激活并执行以下命令：
```bash
python main.py
```

### 主要功能模块：
- **数据下载 (Step A)**：根据指定的配置自动获取最新的医药市场数据，并保存至本地。
- **数据清洗 (Step B)**：处理下载的数据中的异常值、统一格式和列名。
- **数据标准化 (Step 3)**：将数据规范成统一的核心分析标准格式，以便进行深度统计。
- **可视化看板 (Step 4)**：提供交互式图表、预测模型、和各个医药维度的战略数据看板。支持图表导出以及复制到剪贴板。

## 注意事项与常见问题
- 由于部分依赖下载或缓存文件体积可能较大或包含机密数据，包括 `.pkl` 缓存文件、`Cache` 和 `TSM_Downloads` 目录下的动态数据、以及 `*.xlsx` 等中间文件，**均已在 `.gitignore` 中配置为忽略上传**。
- 同事下载代码后，首次执行相关流程即可自动生成对应的缓存和数据目录。
- 如果在数据处理或者看板加载过程中卡住或遇到意想不到的错误，建议通过清理 `Cache` 目录并重新运行程序解决。
